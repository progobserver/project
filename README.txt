1. Программа работает с БД на MySQL, необходимо ее установить. Далее:
--- в appsettings.json необходимо указать свою строку подключения DefaultConnection
--- то же самое сделать в WebhookController в connectionStringBuilder и в DevelopmentController в connectionString
1.2 Если возникнет необходимость работать с другой СУБД (например, PostgreSQL) то необходимо докачать в NuGet необходимые пакеты и внести необходимые изменения в код

2. Ключевая идея заключается в перехвате вебхука после коммита. 
   Автор использовал собственное зеркало Gitea.
   Работа с GitHub должна быть аналогичной. 
   В настройках нужно найти Webhooks и указать там адрес: "ваш Localhost"/api/Report  
   Например:	https://localhost:7268/api/Report 
3. Для перехвата коммита, сообщение в коммите начинать со занчка # и числа, указываюшего на номер задачи
   Например: git commit -m "#1 первый коммит"
